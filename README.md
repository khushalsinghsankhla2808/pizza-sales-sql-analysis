# 🍕 Pizza Sales SQL Analysis

## 📌 Project Overview

This project focuses on analyzing a pizza sales dataset using SQL to extract meaningful business insights. It covers basic, intermediate, and advanced queries to understand sales performance, customer behavior, and revenue trends.

## 🎯 Objectives

* Calculate total orders and total revenue
* Identify the most popular pizza sizes and types
* Analyze order distribution by date and time
* Perform category-wise sales analysis
* Determine top-performing pizzas based on revenue

## 🗂 Dataset Description

The analysis is based on four relational tables:

* **orders**: Contains order ID, date, and time
* **order_details**: Contains quantity and pizza IDs per order
* **pizzas**: Includes pizza size and price
* **pizza_types**: Contains pizza name, category, and ingredients

## 🛠 Tools & Technologies

* SQL (MySQL/PostgreSQL)
* MySQL Workbench / pgAdmin
* GitHub

## 📊 Key Analysis Performed

* Total number of orders placed
* Total revenue generated from pizza sales
* Most common pizza sizes ordered
* Top 5 most ordered pizza types
* Hourly distribution of orders
* Average pizzas ordered per day
* Category-wise performance analysis
* Top 3 pizzas by revenue in each category
* Percentage contribution of each pizza to total revenue
* Cumulative revenue analysis over time

## 🚀 Key Insights

* Identified high-demand pizza categories and sizes
* Found peak order hours for better business planning
* Highlighted top revenue-generating pizza types
* Provided insights for inventory and sales strategy


## 🧠 Conclusion

This project demonstrates strong SQL skills including joins, aggregations, subqueries, and window functions. It showcases the ability to analyze real-world datasets and derive actionable insights useful for business decision-making.
